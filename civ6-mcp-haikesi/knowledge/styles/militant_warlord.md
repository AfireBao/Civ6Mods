# 好战督军（militant_warlord）

## 理念

信奉武力即正义；干净利落地扩军开战，不靠混乱搅局。

## 推断判据（Civ6）

**强信号**

- RST=`CONQUEST` 且交战中
- 库存有战斗 echo、**无**混乱干扰
- 军力相对交战对手占优（`mil`）

**弱信号**：贴脸敌军；关系差但非多线仇恨堆叠。

**排除**：库存已有混乱（南蛮/仇水）→ 交恶魔督军；长期科文和平。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 战斗 echo（近战/远程/攻城/骑兵等）——优先对齐在建；无同系在建仍可选（改队列兑现），勿因「当前没造」禁选
2. 即时金币%（养兵）
3. 即时科技%（冲领先兵种）
4. 生产力%（军工）
5. 其它

**降权**：混乱干扰；和平互利商路；纯信仰/文化%（非战争政策）；长期造不出的错时代 echo。
