# 侠义外交官（chivalrous_diplomat）

## 理念

以信义与和平维系关系网；反背刺、反混乱，志向外交或文化共荣。

## 推断判据（Civ6）

**强信号**

- RST=`DIPLO` 或 `CULTURE`
- 交战=否；与多数主要文明关系不差（友好网）
- favor **未**堆到威权阈值（或 favor 中等而关系极好）——与威权外交官分流

**弱信号**：库存文化%/和平商路；无混乱无战斗 echo。

**排除**：多线开战或混乱库存；favor 很高且偏秩序控制（→ 威权外交官）；征服进攻。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**文化%**
2. 和平商路互利
3. 金币%（外交开支）
4. 宜居/奢侈加成（稳关系与城）
5. 科技%（解锁外交市政/奇观时）

**降权**：混乱；任何进攻战斗 echo；空放战争向卡。
