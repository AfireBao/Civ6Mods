# 谋略督军（strategist_warlord）

## 理念

以科技武装战争机器；开战也要兵种代差，不作无脑莽冲。

## 推断判据（Civ6）

**强信号**

- 交战中，且 RST=`SCIENCE` 或科技产出突出
- 库存科技% 与/或战斗 echo
- 少混乱、非纯和平发育

**弱信号**：CONQUEST + 高科；被动防守升级军种。

**排除**：长期和平无威胁（→ 博学贤者）；混乱库存为主（→ 恶魔督军）；高信仰宗教主建。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**科技%**
2. 战斗 echo（兑现代差）
3. 生产力% / 工人 echo（军工与学院）
4. 金币%（维护与加速）
5. 其它

**降权**：混乱干扰；无交战/无贴脸威胁时仍可拿科技%，但战斗 echo 次于科技。
