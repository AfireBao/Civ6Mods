# 竞争商人（competitive_merchant）

## 理念

精于算计的金权玩家；商路与金币第一，奇观与浪漫事业靠后。

## 推断判据（Civ6）

**强信号**

- 商路活跃 + 金币产出/库存突出
- RST 非征服开战主线；不以高信仰或高 favor 外交为主

**弱信号**：和平；科文 RST 但金远高于奇观/工人信号。

**排除**：CONQUEST 交战 + 混乱/战斗 echo 堆；工匠向（高产工人+科文且金一般）→ 工匠商人；高 favor 外交官。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**金币%**
2. 商路互利（有入向/出向时）
3. 生产力%（赚得更快）
4. 科技或文化%（次于金钱）
5. 其它

**降权**：混乱；进攻战斗 echo；纯信仰%；为奇观而奇观的工人空放（次于金）。
