# 狂热隐士（fanatic_isolationist）

## 理念

以内向信仰凝聚力量；偏好宗教胜利或征服胜利，少依赖外贸与外交合纵。

## 推断判据（Civ6）

**强信号**

- RST=`RELIGION` 或 `CONQUEST`
- 信仰产出（`faith`）相对突出
- 库存含信仰%、宗教相关短名，或战斗 echo（征服支线）
- 商路少：capacity/active 低或国际路少（隐士）

**弱信号**：交战中（征服支线）；文化/外交 RST 弱。

**排除**：RST=`DIPLO` 且 favor 高、商路繁忙的和平外交型；或明确商人向（高商路+金币生产力%、无信仰）。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**信仰%**
2. 宗教/信仰向资源创建或相关（若候选有）
3. 若 RST 偏征服或已交战：战斗 echo / 混乱干扰
4. 食物/生产力%（内政自给）
5. 其它

**降权**：和平互利国际商路（天朝/两河）——违背隐士；纯外交/使者向（若出现）；无战事时的纯军事 echo 可次于信仰。
