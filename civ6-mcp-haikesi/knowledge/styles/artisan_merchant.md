# 工匠商人（artisan_merchant）

## 理念

重生产与金币，商路兴旺，好造奇观；志向科技或文化长线，少靠蛮力开战。

## 推断判据（Civ6）

**强信号（≥2 且总分≥3）**

- RST=`SCIENCE` 或 `CULTURE`
- 商路容量/已用较高，或国际入向/出向≥1（`trade`）
- 库存含生产力%/金币%、两河/天朝等商路互利，或工人 echo
- 产出上金或产相对突出（相对人口）

**弱信号**：和平（交战=否）；在建奇观/剧院/工业（`city_pressure` 文案含奇观名时，仅作弱信号）。

**排除**：RST=`CONQUEST` 且交战中并已有混乱/大量战斗 echo（更偏恶魔督军）。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**生产力%**、工人/建造者 echo（加速奇观与基建）
2. 商路互利（两河粮仓、天朝上国等）——有入向或可预期时优先
3. 即时**金币%**（养商路与维护）
4. 即时科技% 或 文化%（服务科文胜利）
5. 资源创建（地块产/宜居）填板

**降权**：混乱干扰；纯军事 echo（非防守必要）；攻城/骑兵翻倍开战向。
