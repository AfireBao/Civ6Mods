# 威权外交官（authoritarian_diplomat）

## 理念

以秩序与宗主权扩展影响力；偏好外交或文化胜利，多交好主要文明与城邦，用使者与外交资本取胜。

## 推断判据（Civ6）

**强信号**

- RST=`DIPLO` 或 `CULTURE`
- 外交赞同（`favor`）明显积累，或已接触多位主要文明且多数和平、关系不差
- 库存含文化%、商路互利（宗主/同盟友好向）、少混乱少战斗 echo

**弱信号**：交战=否；文化产出偏高；科技非主战略。

**排除**：多线交战或 RST=`CONQUEST` 且库存混乱/战斗 echo 为主。

**说明**：城邦宗主/使者数量若 CTX 未直接给出，用 favor + 和平关系网 + CULTURE/DIPLO 作代理信号；后续 gather 补使者字段后可加强判据。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**文化%**（政策、使者相关市政与城邦竞争）
2. 和平互利商路（两河/天朝等）——巩固友好与收益
3. 即时金币%（买使者/外交开支的间接支撑）
4. 即时科技%（解锁外交奇观/市政所需科技时抬权）
5. 宜居/奢侈创建（稳城，少叛乱）

**降权**：混乱干扰（南蛮/仇水）；进攻向战斗 echo；无入向时的空放商路卡仍可按风格硬选（cosplay），但注明可能空放。

**后续海克斯**：使者、城邦、外交奇观、赞同、世界议会相关词条优先挂 `preferred_by: authoritarian_diplomat`。
