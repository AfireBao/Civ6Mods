# 博学贤者（erudite_sage）

## 理念

以智慧与科技取胜；偏好科技胜利，厌恶主动开战。只有面临**被宣战/交战危机**时，才考虑战斗兵种 echo 自保。

## 推断判据（Civ6）

**强信号**

- RST=`SCIENCE`
- 科技产出（`sci`）相对突出，或库存已有科技%
- 交战=否，或仅被动防守（无多线开战、无混乱库存）

**弱信号**：文化次高；商路有但非主；favor 中等。

**排除**：RST=`CONQUEST` 且交战进攻中并持混乱/大量进攻 echo；或 RST=`RELIGION` 信仰主建。

## 风格 Skill（仅 cosplay）

不受 `_payoff` 管理；须守合法性底线。

偏好序：

1. 即时**科技%**
2. 生产力% / 工人 echo（加速学院与太空基建）
3. 茶等**科技向**资源创建（有城时）
4. 金币%（买研究加速/维护学院城市）
5. 其它和平发育

**战斗 echo（严格）**：

- **默认不选**近战/远程/攻城/骑兵等战斗翻倍。
- **仅当**极短况 `交战=是`，或 `border_threats` 显示交战敌方贴脸单位时，才允许选战斗 echo 自卫。
- 即使如此，仍优先能立刻产线兑现的防御向单位；混乱干扰（南蛮等）降权（贤者不主动搅局），除非明显只打敌人且己方安全。

**降权**：主动战争向混乱；无危机时的任何战斗 echo；信仰主建（除非科技附带）。
