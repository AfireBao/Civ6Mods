# 恶魔督军（demonic_warlord）

## 理念

信奉混沌理念，希望为世界带来混乱与战争；与多数领袖关系差。  
（叙事用；局内 AI 行为不因本标签改变。）

## 推断判据（Civ6）

**强信号（≥2 → 高置信贴标）**

- RST=CONQUEST（征服策略优先）
- 交战中，且边境可见敌方军事单位
- 对多位主要文明关系差 / 不满高 / 负面外交修饰多
- 库存已有混乱干扰或大量战斗 echo

**弱信号**：军力压制交战对手；金币紧但仍狂造军。

**排除**：长期和平的科文向 + 库存以商路互利/纯发育%为主。

## 风格 Skill（仅 cosplay 掷骰时注入）

本篇**不受**收益优先（`_payoff`）管理；仍须遵守合法性底线（候选 ID 等）。

偏好序：

1. 混乱干扰类型（南蛮、仇水等）——优先制造全图麻烦
2. 战斗 echo（近战/远程/攻城/骑兵等）——扩张战争机器
3. 即时金币%（养兵）
4. 即时科技%（冲领先兵种）
5. 其它

**降权（风格偏好，非收益硬规则）**：和平互利商路卡；纯信仰/文化%（除非明显服务战争政策）。
